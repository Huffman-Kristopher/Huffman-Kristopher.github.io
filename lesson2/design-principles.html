Design Principles

Kristopher Huffman

White Space and Clean Design

From Microsoft's xbox.com

On the Xbox homepage there is plenty of white space between objects. This allows for the page to be viewed easy and content to be separated in a way that is pleasant to the user. The user is able to find where their attention should be focused.

Contrast

From jurassicworld.com

Great use of contrast in the site. The fonts are contrasting so the it is easy to see what the topics and content are for each area. The colors also contrast so the users eyes are not strained while navigating the page.

Repetition

From omsi.edu

The Oregon museum of Science and Industry is a good example of repetition. The layout is the same from page to page, making for a cohesive user experience. The colors, fonts, and alignments are consistent across the website. This allows the user to have a good visit without a headache.
